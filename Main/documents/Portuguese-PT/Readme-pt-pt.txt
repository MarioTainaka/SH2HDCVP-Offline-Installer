=====================================================

Silent Hill 2 HD Collection Voice Pack Version 5.0.5
=====================================================

O "Audio Enhancement Pack" e o "FMV Enhancement Pack" NÃO são necessários para que este mod funcione, pois estes já são utilizados como base.



=================================
Instalação
=================================

1:	Transfere e executa o instalador do Silent Hill Enhanced Edition a partir do link abaixo. Executa o instalador e instala todos os pacotes, exceto o FMV Enhancement Pack e o Audio Enhancement Pack.

2: 	Transfere e executa o instalador do Silent Hill 2 HD Collection Voice Pack. O instalador vai apontar para o caminho de instalação do Silent Hill 2, que, por predefinição, estará instalado em C:\Program Files (x86)\KONAMI\Silent Hill 2. (Não é recomendado que o jogo esteja instalado aqui.) 

3:   	Instala o HD Collection Voice Pack.

4:    	Executa o jogo.

5:	Os créditos, histórico de alterações e este ficheiro "readme" estão localizados em "Konami\Silent Hill 2\HD Collection Voice Pack Support".



==================================================
Componentes da Instalação
==================================================

Main: This component includes the Enhanced Edition FMV,s, Sound Effects and the required Dialogue files. This is required for the mod to work and cannot be unchecked

Cutscene Subtitle Fixes For All Languages: There are two in-game cutscenes which had their context changed massively in the HD Collection due to oversights. This option fixes the subtitles to accommodate for this. By default this is fixed when playing the game in English only. This options enables this fix for all languages. It is not recommended to enable this fix if you're using translation packs as the files will be overwritten.

Audio Enhancement Pack Music: This option reinstalls the Enhanced Edition music files included in the Audio Enhancement Pack, If you're running the game from a Harddrive or SD card it is highly recomended to enable this options as the game will run into performance issues.


==================================================
Desinstalação\ Atualização\Desativação do Mod
==================================================

1:	Remove a versão antiga a partir da opção "Adicionar ou remover programas" do Windows, e instala a versão mais recente. Caso queiras, podes atualizar a versão Main para Lite, e vice versa.

2:      Se pretendes desativar o mod sem o desinstalar, podes mudar o nome da pasta "lang" para qualquer outra coisa (ex. "lang_backup). Quando desinstalares o mod, não te esqueças de voltar a mudar o nome da pasta para o original!




Este mod é um projeto de paixão que substitui as vozes originais do jogo pelas vozes da versão HD de 2012, gravadas no estúdio PCB de Las Vegas, famoso por dobrar os teus jogos favoritos como a saga Persona. O mod também inclui o FMV Enhancement Pack, que recodifica as sequências FMV de maneira a utilizar as vozes da versão HD. Estas vozes foram também remisturadas para 2 canais (stereo) ao invés de 5.1, e todos os erros de mistura foram corrigidos.
